# 🔳 QR Code Generator

A simple and fast QR Code Generator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Generate QR codes instantly
- Supports text and URLs
- Clean and simple UI
- Works offline (no API required)

## 📂 Project Structure
```
├── index.html
├── styles.css
├── script.js
├── qrious.min.js
```

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- QRious.js Library

## ▶️ How to Use
1. Open `index.html` in your browser
2. Enter text or URL
3. Click **Generate QR Code**
4. Your QR code will be displayed instantly

## 🔮 Future Improvements
- Download QR Code as image
- Change QR size dynamically
- Dark mode support
